Установка Django:

  1. cmd - заходим в командную строку
  2. pip3 install django - устанавдиваем в винду
  3. python.exe -m pip install --upgrade pip - обновляем
  4. переход в папку проекта (в командной строке) – т.е куда проект установить (админку, manage.py и тд)
  5. django-admin startproject myfirstproject - создание нового проекта с названием

Команды командной строки:

  • dir – посмотреть что в папке
  • cd – следовать в какую-то папку (cd название папки)
  • cls – очистить консоль
  • ctrl + c – остановить runserver
  • .. - вернутся в дирректорию ниже

Старт проекта Django:

  1. cmd - заходим в командную строку
  2. заходим в корневую папку проекта (в cmd)
  3. python manage.py runserver - старт локального сервера
    http://127.0.0.1:8000/ - это мой свервер.
  4. python manage.py help - команды manage.py помощь
  5. Ctrl + C - остановить локальный сервер

GIT

Git - тип системы контроля версии
например если редактируем файл, Git помогает отслеживать: что было изменено, кто изменил и зачем..

  1. C:\Program Files\Git\git-cmd.exe - cmd git
  2. В командной строке переходим в корневую папку проекта
  3. git –v - посмотреть версию git
  4. git init – устанавливвем гит в папку проекта
  5. git status – смотрим что отслеживает или неотслеживает
  6. git add . – добавить чтобы отслеживал всё в папке
  7. git config --global user.email elproject220@yandex.ru
  8. git config --global user.name "Mello"
  9. git commit -m "Имя коммита1" - создали коммит1
  10. git status - после создания первого коммита изменений не было
  11. изменяем что то в проекте
  12. git status - указывает что изменилось
  13. git add . – опять добавили в гит (все)
  14. git commit -m "Имя коммита2" - создали коммит2
  15. Залогинились в гитхаб.ком, создали в репозитории новый проект
  16. git remote add origin https://github.com/Mello134/viceversa.git - отправили в репозиторий гитхаб весь проект
  17. git push -u origin main
    Этот и предыдущий пункт уточняй в самом репозитории в проекте, там есть коды их можно просто скопировать!
  18. Чтобы скачать с репозитория. Заходим в cmd в нужную папку, далее
    git clone https://github.com/Mello134/viceversa.git - скачали commit с своего репозитория (ссылка там)

Доп комманды git:

  • git log –-oneline - список всех коммитов в папке
